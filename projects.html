RDF Research

Over the summer of 2015, I worked for Dr. Marek Reformat testing a new library for his RDF research. Dr. Reformat would gave me simple tasks weekly or biweekly that would test the aspects of the library he needed. In this way, I slowly learned Java basics and a lot about RDF data and using data!

RDF stands for Resource Description Framework - a model for describing data on the Web. A RDF triple is a unit of RDF data - it's called a triple because its form consists of three parts: a subject, a property and an object. Each triple individually describes the subjects relationship with the object and with an entire set of triples, you can create a graph (the subject and object are nodes, property is an edge) and therefore describe the relationship among many subjects/objects!

Review Template App

The idea behind this project comes from the realization I had that people's opinions are the best form of qualitative data for many things in our world. Applications like Rotten Tomatoes and Yelp have become very popular offering this simple service to a topic of need (movies and restaurants respectively). This project is my attempt to create such an application for a now unknown/undecided topic of need.

The template uses ASP.NET's MVC framework and Entity Framework. I used .NET's scaffolding to quickly create the front end and used Bootstrap for some looks. Entity Framework took care of all the object-relational mapping - what data goes where and is related to other data how. I based this project on a tutorial from ASP.NET's website, it can be found here.

Conference App

I built a mobile Confrenece app while I worked for fourthdimension inc. as an intern. The mobile webapp was created using the Ionic Framework and Apache Cordova. The app was built to provide attendees with the conference schedule, details on the seminars and presenters, maps of the area and provided internal staff with user feedback on seminars.

The application handled all this data with read and write capabilities to an Azure Mobile Service and some JavaScript services. I worked with coworkers to create the user interface of the application, which was built with SCSS and CSS.

The project required me to learn and deal with Angular.js framework, Ionic framework, Command Line, GIT, CSS some JavaScript and Azure Databases. It taught me lessons about working with a team in an office environment. Unfortunately, since the application was created under contract, I do not have access to point you to any of the work I did.
